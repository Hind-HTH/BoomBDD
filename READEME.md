# app_ecommerce_bachelor_3

# Documentation APP BONUS

## Installation
Pour lancer le server api
```
cd api 
npm install
npm run dev
```
Pour lancer l'application
```
cd frontend
npm install
npm run dev
```


# Links

### Prod link
* http://localhost:5000/api/v1/{ressource}/{action}
* exemple: http://localhost:5000/api/v1/student/get-all-student
